# Análisis de Ventas AdventureWorks - SQL & Power BI

## 📝 Descripción
Este proyecto analiza el rendimiento de ventas de la empresa ficticia AdventureWorks. Se enfoca en KPIs financieros y en la retención de clientes.

## 🛠️ Herramientas Utilizadas
* **SQL Server:** Extracción y transformación de datos (ETL).
* **Power BI:** Modelado de datos y visualización.
* **DAX:** Creación de medidas para métricas de negocio.

## 📊 KPIs Principales
* **Ingresos Totales:** Suma de ventas netas.
* **Tasa de Retención:** Porcentaje de clientes con 2 o más compras.
* **Ticket Promedio:** Valor medio de venta por factura.

## 🚀 Cómo ver el proyecto
1. Puedes ver la imagen del dashboard en la carpeta raíz.
2. El archivo `.pbix` contiene el reporte interactivo.
3. El archivo `queries.sql` contiene la lógica de extracción de datos.
